# Observability

## Required Signals

- request id;
- model used;
- tokens and cost;
- latency;
- tool calls;
- eval outcome;
- validation risk counts;
- retrieval source count;
- error codes;
- user feedback.

## Tools

- Langfuse for AI traces;
- Sentry for errors;
- OpenTelemetry for backend traces;
- PostHog for product analytics;
- Grafana/Prometheus for infrastructure.
