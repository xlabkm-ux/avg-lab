## Путь пользователя

### Точка входа
- Пользователь находится в активном рабочем пространстве проекта
- У пользователя были диалоговые обмены и/или сессии обоснованного поиска
- Пользователь нажимает вкладку «Артефакты» в навигации

### Основная концепция
Артефакты — это **экспортируемые снимки** работы мышления пользователя:
- **Сводки сессий** — Обзор всех обменов в сессии
- **Отчёты о цитировании** — Скомпилированный список всех использованных цитат
- **Снимки карты** — Состояние графа в определённый момент времени
- **Разности карт** — Изменения между двумя состояниями графа

Эти артефакты можно сохранять, передавать другим и использовать как результаты процесса мышления.

### Порядок действий
1. Пользователь выбирает вкладку «Артефакты»
2. Пользователь видит поверхность артефактов (в настоящее время заполнитель)
3. Пользователь может просмотреть доступные типы артефактов
4. Пользователь выбирает артефакт для экспорта
5. Система генерирует артефакт
6. Пользователь загружает или копирует артефакт

## Сценарии тестирования

### TC-008.1: Отображение вкладки «Артефакты» (текущее состояние)
**Предусловие**: Пользователь находится в активном рабочем пространстве проекта

**Шаги**:
1. Нажмите вкладку «Артефакты» в навигации
2. Убедитесь, что поверхность артефактов отображается
3. Убедитесь, что отображается содержимое-заполнитель: «Экспорт сводок сессий, цитат и снимков карты»

**Ожидаемые результаты**:
- Вкладка «Артефакты» выбрана и активна
- Отображается текст-заполнитель
- Ошибок в консоли нет
- Навигация по вкладкам работает

**Приоритет**: P0

### TC-008.2: Экспорт сводки сессии (будущее)
**Предусловие**: Поверхность артефактов реализована

**Шаги**:
1. Перейдите на вкладку «Артефакты»
2. Найдите опцию экспорта «Сводка сессии»
3. Выберите эту опцию
4. Настройте параметры экспорта (если применимо):
   - Диапазон дат
   - Включить/исключить метаданные
   - Формат (JSON, Markdown, PDF)
5. Нажмите «Экспорт»
6. Убедитесь, что сводка генерируется

**Ожидаемые результаты**:
- Экспорт генерирует полную сводку сессии
- Включает все диалоговые обмены
- Включает все сгенерированные утверждения
- Включает все использованные цитаты
- Формат читаемый и структурированный

**Приоритет**: P1

## Модели данных

### GraphSnapshot (Снимок карты)
```typescript
interface GraphSnapshot {
  nodes: AvgMapNode[];
  edges: AvgMapEdge[];
}
```

### GraphDiff
```typescript
interface GraphDiff {
  addedNodes: AvgMapNode[];
  updatedNodes: AvgMapNode[];
  removedNodeIds: string[];
  addedEdges: AvgMapEdge[];
  updatedEdges: AvgMapEdge[];
  removedEdgeIds: string[];
}
```

### MapDiffArtifact
```typescript
interface MapDiffArtifact {
  kind: "map_diff";
  from: GraphSnapshot;
  to: GraphSnapshot;
  diff: GraphDiff;
}
```

### SessionSummary (будущее)
```typescript
interface SessionSummary {
  project_id: string;
  session_id: string;
  created_at: string;
  messages: DialogueMessage[];
  claims: AvgClaim[];
  citations: AvgCitation[];
  map_snapshot?: GraphSnapshot;
}
```

### CitationReport (будущее)
```typescript
interface CitationReport {
  project_id: string;
  created_at: string;
  citations: AvgCitation[];
  retrieval_confidence: "none" | "low" | "medium" | "high";
  source_documents: string[];

```

| Сценарий | Ожидаемое поведение |
|----------|-------------------|
| История сессий отсутствует | Показать заполнитель с рекомендациями |

| Загрузка файла заблокирована браузером | Показать опцию ручного копирования |
