# UC-007: Рецензирование цитат

## Обзор

| Поле | Значение |
|-------|-------|
| **ID** | UC-007 |
| **Название** | Рецензирование цитат |
| **Описание** | Пользователь просматривает, фильтрует и взаимодействует с цитатами из доказательной базы, полученными в результате поиска с обоснованием |
| **Актор** | Пользователь, выполнивший запрос поиска с обоснованием |
| **Приоритет** | P0 - Критический |
| **Статус** | Реализовано |
| **Связанный E2E** | `e2e/tests/citation-panel.spec.ts` |

---

## Путь пользователя

### Точка входа
- Пользователь выполнил запрос поиска с обоснованием (UC-004)
- Поиск вернул соответствующие фрагменты доказательств
- Панель цитат отображается как часть результатов поиска

### Основная концепция
Цитаты — это **каркас доказательств** обоснованного поиска AVG. Каждая цитата:
- Связывает утверждение с источником доказательств
- Показывает уровень уверенности и оценку релевантности
- Предоставляет точный найденный текст
- Обеспечивает проверку и отслеживаемость

Пользователи могут просматривать, фильтровать и копировать цитаты, чтобы убедиться, что ответы должным образом обоснованы.

### Поток действий
1. После отправки запроса с обоснованием пользователь видит Панель цитат
2. Панель отображает количество результатов: "N результатов"
3. Пользователь может фильтровать цитаты по уровню уверенности (Все/Высокий/Средний/Низкий)
4. Каждая цитата показывает:
   - Метка источника
   - Значок уверенности
   - Оценка релевантности
   - ID фрагмента, ID цитаты, ID документа
   - Найденный текст (обрезан до 150 символов)
5. Пользователь нажимает на цитату, чтобы развернуть полный найденный текст
6. Пользователь может скопировать текст фрагмента в буфер обмена из развернутого вида

**Ожидаемые результаты**:

### TC-007.2: Отображение значка уверенности
**Предусловие**: Цитаты отображаются

**Шаги**:
1. Найдите значок уверенности на каждой цитате
2. Убедитесь в уровне уверенности и цвете:

| Уверенность | Цвет | Значение |
|------------|-------|---------|
| `high` | Зелёный | Сильное совпадение, высокая релевантность |
| `medium` | Жёлтый | Умеренное совпадение, частично релевантно |
| `low` | Красный | Слабое совпадение, низкая релевантность |

**Ожидаемые результаты**:
- Значок отображает правильный уровень уверенности
- Цветовая кодировка соответствует уверенности
- Значок хорошо виден на каждой цитате

**Приоритет**: P0

## Конечные точки API

Цитаты возвращаются как часть ответа обоснованного поиска, а не через отдельную конечную точку.

### Ответ поиска
```json
{
  "hits": [
    {
      "snippet_id": "snip-1",
      "document_id": "doc-abc",
      "project_id": "proj-xyz",
      "score": 0.92,
      "confidence": "high",
      "citation_id": "cit-1",
      "matched_text": "Employees may work from home up to 3 days per week...",
      "source_label": "Remote Work Policy"
    }
  ],
  "retrieval_confidence": "high"
}
```

## Модели данных

### AvgRetrievalHit
```typescript
interface AvgRetrievalHit {
  snippet_id: string;
  document_id: string;
  project_id: string;
  score: number;                    // Оценка релевантности (0-1)
  confidence: "low" | "medium" | "high";
  citation_id: string;
  matched_text: string;
  source_label: string;
}
```

### AvgCitation
```typescript
interface AvgCitation {
  id: string;
  document_id: string;
  snippet_id: string;
  source_label: string;
  quoted_text: string;
  relevance: "supporting" | "context" | "contradicting";
}
```

## Обработка ошибок

| Сценарий | Ожидаемое поведение |
|----------|-------------------|
| Копирование не удалось (буфер обмена недоступен) | Показать уведомление об ошибке, разрешить ручное выделение |
| В данных цитаты отсутствуют поля | Показать частичные данные, записать предупреждение |
| Очень длинный найденный текст (>10 КБ) | Обрезать с предупреждением, показать "Текст обрезан из-за длины" |
