# User Personas

## 1. Founder / Product Creator

Needs:

- turn a vague product idea into a concept;
- test differentiation;
- prepare pitch, roadmap, feature map.

AVG value:

- creates options;
- attacks weak assumptions;
- makes the concept investor/product ready.

## 2. Writer / Thinker

Needs:

- structure raw ideas;
- keep depth without fog;
- create essays, books, lectures.

AVG value:

- maps arguments;
- marks metaphors;
- produces clean drafts.

## 3. Researcher / Methodologist

Needs:

- compare concepts;
- track hypotheses;
- avoid category mistakes.

AVG value:

- graph map;
- claim status;
- source and assumption tracking.

## 4. Team Facilitator

Needs:

- run creative sessions;
- avoid groupthink;
- capture decisions.

AVG value:

- multiple modes;
- shared artifacts;
- decision ledger.
