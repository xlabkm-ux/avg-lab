# Context Pack: Product

AVG is a creative dialogue environment, not a chatbot wrapper.

Core jobs:

1. Generate idea variants.
2. Build a concept map.
3. Validate claims.
4. Detect weak or inflated language.
5. Turn dialogue into artifacts.

Primary modes:

- Разгон / Creative expansion.
- Карта / Map building.
- Сократ / Socratic questioning.
- Архитектор / System design.
- Провокатор / Critique.
- Редактор / Artifact editing.
- Валидатор / Claim safety.
